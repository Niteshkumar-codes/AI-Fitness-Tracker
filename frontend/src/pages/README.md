# Pages Directory

This directory contains the main page views of the application. Each subfolder represents a feature module or route:

- **`auth/`**: Login, Register, Forgot Password pages.
- **`dashboard/`**: Core user landing dashboard showing quick stats.
- **`workout/`**: Workout listing and logging views.
- **`food/`**: Nutrition/Diet logging and dashboard.
- **`goals/`**: Target weight and calorie goal configurations.
- **`water/`**: Water logging and history charts.
- **`profile/`**: User settings and profile updates (height, weight, age, name).
