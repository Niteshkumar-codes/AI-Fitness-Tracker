# Components Directory

This directory contains reusable UI components for the React application. It is divided into three key subfolders:

- **`common/`**: Application-specific reusable components used across multiple pages (e.g., Header, Footer, Sidebar, Navigation bar).
- **`dashboard/`**: Special sub-components that belong specifically to the dashboard page layout (e.g., StatsCard, Charts, DailySummary).
- **`ui/`**: Generic, low-level component design tokens (sometimes called atomic UI elements) that have no specific business logic, such as Button, Input, Modal, Badge, and Spinner.
